# Dotfiles

### Description

- **Protocol:** Wayland
- **Windows Manager:** Sway
- **Terminal:** Alacritty
- **Launcher:** Alacritty + fzf
- **Notifications:** Maco
- **Progress Bar:** Wob
- **Font:** Terminus
